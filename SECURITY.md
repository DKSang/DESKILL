# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 2.x     | Yes       |
| 1.x     | No        |

## Reporting a Vulnerability

This repository contains skill definitions and educational content — no production code, no credentials, no secrets.

If you find a security issue (e.g., a template that suggests insecure practices, or a pattern that could lead to data exposure), please open an issue with the `security` label.

Report sensitive issues privately by emailing the repository owner.
